# Design Context — Aureum App

## Users
- Primary user: Guilherme (founder, operator). Power user who lives in the app daily across client funnels, content, ads, CRM, and finance.
- Context: running a funnel agency for creators (e.g. Luke Alexander · AI Insiders). Reviews each client's full funnel at a glance, checks ads, message sequences, tags, and metrics.
- Job to be done: see the complete state of a client's funnel — creative, pages, automations, destinations, money — without digging through 5 SaaS tools.

## Brand Personality
- Three words: **disciplined, calm, operator-grade**.
- Emotional goal: feels like a high-signal operations console, not a marketing template.
- Anti-references: "clean SaaS pastel dashboards," confetti-and-emoji playful, anything neon-on-dark AI-generated.

## Aesthetic Direction
- **Dark, uniformly.** No light-mode toggle planned.
- **Minimal color.** Neutrals first. One accent only: muted green for money/positive metrics (CPL, revenue, ROAS, conversion %).
- Brand colors (Meta blue, Telegram blue) only allowed **inside tiny 12–14px icon badges** for recognition — never as card borders, backgrounds, or strokes.
- Authentic client-surface colors are welcome **inside message-preview cards** (Gmail white, iMessage black, Telegram navy) because they signal "this is what the recipient actually sees."
- Typography: Geist Variable is already loaded and fine to use — it's neutral enough to not fight the dark chrome. Monospace only for code-like values (tag slugs, API actions).

## Design Principles
1. **The grid is the UI.** Strict column and row alignment communicates structure. When spacing varies, it varies *for a reason* (hierarchy).
2. **Money is the only color.** Accents are reserved for financial/positive signal. Everything else wears neutral.
3. **Mimic the destination.** When rendering a message, match the real client (Gmail, iMessage, Telegram) so users recognize the form immediately. Outer chrome stays Aureum-neutral.
4. **Dense but breathing.** This is a data-dense operator tool — tight where related, generous between groups. Never empty for its own sake.
5. **No decoration.** No gradient text, no border-left accent stripes, no glassmorphism, no sparklines-as-garnish, no rainbow edge colors. If it isn't information, it isn't on screen.
